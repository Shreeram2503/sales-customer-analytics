# Sales & Customer Analytics Project

This is an end-to-end data analytics project using a real-world e-commerce dataset.

## Dataset
- Source: Brazilian E-Commerce Public Dataset (Olist) from Kaggle
- The dataset is not uploaded to this repository due to size limitations.
- Data is loaded locally for analysis.

## Objectives
- Analyze sales performance
- Understand customer behavior
- Identify product trends
- Derive actionable business insights

## Tools
- Excel
- Python
- SQL
- Data Visualization
